Connect Form to Google Sheets

A free, no-code setup using Google Apps Script. Takes about 10 minutes.

Step 01
Create your Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it "ONO Mentor — Assessment Leads".

In Row 1, add these exact column headers (one per cell, A to I):

Timestamp  |  First Name  |  Last Name  |  Phone  |  Email  |  Exam  |  Class  |  Coaching  |  Challenge
Step 02
Open Apps Script

In your Google Sheet, click Extensions → Apps Script. A new tab will open with the script editor.

Delete any existing code and paste the following script entirely:

function doPost(e) {
  const sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();

  let data;
  try {
    data = JSON.parse(e.postData.contents);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'error', message: 'Invalid JSON' }))
      .setMimeType(ContentService.MimeType.JSON);
  }

  sheet.appendRow([
    data.timestamp   || new Date().toLocaleString(),
    data.fname       || '',
    data.lname       || '',
    data.phone       || '',
    data.email       || '',
    data.exam        || '',
    data.cls         || '',
    data.coaching    || '',
    data.challenge   || ''
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({ status: 'success' }))
    .setMimeType(ContentService.MimeType.JSON);
}

Click the 💾 Save icon (or Ctrl+S).

Step 03
Deploy as a Web App

Click the blue Deploy button (top right) → New deployment.

  1. Click the gear icon next to "Select type" → choose Web app
  2. Description: ONO Mentor Form
  3. Execute as: Me
  4. Who has access: Anyone
  5. Click Deploy
  6. Authorize the app when prompted (click "Allow")
Important: After deploying, you'll see a "Web app URL" that looks like:
https://script.google.com/macros/s/AKfycb.../exec
Copy this URL — you'll need it in the next step.
Step 04
Paste the URL into ono-mentor.html

Open ono-mentor.html in a text editor (VS Code, Notepad++, etc.).

Find this line near the bottom of the file (inside the <script> tag):

const SHEET_URL = 'YOUR_APPS_SCRIPT_URL_HERE';

Replace YOUR_APPS_SCRIPT_URL_HERE with your copied URL:

const SHEET_URL = 'https://script.google.com/macros/s/AKfycb.../exec';

Save the file. That's it — your form will now write every lead directly to Google Sheets.

Step 05
Test the connection

Open ono-mentor.html in your browser, fill in the assessment form, and submit it. Then check your Google Sheet — a new row should appear within 2–3 seconds.

If rows aren't appearing: Go back to Apps Script → Deploy → Manage deployments → make sure "Who has access" is set to Anyone. Then re-deploy with a new version.
✓ Setup Complete
You're live.

Every assessment booking from your website will now appear as a new row in your Google Sheet with full details: name, phone, email, exam, class, coaching institute, and timestamp.

Tip: In Google Sheets, go to Tools → Notification rules to get an email alert every time a new lead submits the form.
